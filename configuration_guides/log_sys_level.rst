*************************
Determining Client Level
*************************
The ``logSysLevel`` flag determines the client log level, as follows:

* **0** - L_SYSTEM
* **1** - L_FATAL
* **2** - L_ERROR
* **3** - L_WARN
* **4** - L_INFO
* **5** - L_DEBUG
* **6** - L_TRACE

The following describes the ``logSysLevel`` flag:

* **Data type** - uint
* **Default value** - ``100000``
