*************************
Setting InMemory Spool Memory
*************************
The ``cacheRamGB`` flag sets the amount of memory (GB) to be used by Spool InMemory.

The following describes the ``cacheRamGB`` flag:

* **Data type** - size_t
* **Default value** - ``16``
* **Allowed values** - 0-4000000000
