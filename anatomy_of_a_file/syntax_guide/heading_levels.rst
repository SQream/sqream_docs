***********************
Setting Heading Levels
***********************
The following shows how to set heading levels:

.. code-block:: console

   Heading 1
   =====================

   Heading 2
   ---------------------

   Heading 3
   ~~~~~~~~~~~~~~~~~~~~~

   Heading 4
   *********************

.. note:: You can use any heading syntax for any level as long as you use them consistently per page.
