.. _extra_lines:

***********************
Inserting Extra Lines
***********************
You can space your bullets and numbered list items out as follows:

.. code-block:: console

   * Bullet 1

      ::

   * Bullet 2s

      ::

   * Bullet 3

Using the ``::`` syntax spaces your bullets and numbered lists out as follows:

* Bullet 1

   ::

* Bullet 2

   ::

* Bullet 3
