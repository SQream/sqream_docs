***********************
Exporting Data
***********************  
You can export data from SQream, which you may want to do for the following reasons:


* To use data in external tables. See `Working with External Data <https://docs.sqream.com/en/v2021.2/operational_guides/external_data.html>`_.
* To share data with other clients or consumers with different systems.
* To copy data into another SQream cluster.

SQream provides the following methods for exporting data:

* Copying data from a SQream database table or query to another file - See `COPY TO <https://docs.sqream.com/en/v2021.2/reference/sql/sql_statements/dml_commands/copy_to.html>`_.
